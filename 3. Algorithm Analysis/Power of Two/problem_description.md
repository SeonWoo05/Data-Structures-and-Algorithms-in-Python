# 2의 제곱수
주어진 수가 2의 제곱수 인지를 확인합니다.

## 문제 풀이 설명
### 1. 2로 반복해서 나누어 1이 되는지 확인
• `n = 0` 인 경우 먼저 제외  
• 2로 나누어지지 않을 때까지 나눔

```python
if n == 0:
        return False
    while n % 2 == 0:
        n = n/2
    if n == 1:
        return True
    else:
        return False
```

### 2. 비트 조작
• 2의 제곱수는 비트로 표현하면 1이 하나만 존재  
• `n`이 2의 제곱수라면, `n-1`과 비트 AND 연산을 하면 0이 됨

## 알고리즘의 시간복잡도
• 1.$T(n) = O(lg n)$  
• 2.$T(n) = O(1)$
